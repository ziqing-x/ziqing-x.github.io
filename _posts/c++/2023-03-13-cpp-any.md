## 一、简介
std::any 是 c++17 中引入的一个类模板，它允许存储和检索任意类型的值，提供了一种类型安全的、动态的值存储和访问机制。

## 二、例子
```c++
#include <iostream>
#include <any>

struct Person
{
    std::string name;
    int age;
};

int main()
{
    std::any var = 88;
    std::any str = "hahaha";
    Person p = {
        "老王",
        88
    };
    std::any info = p;

    std::cout << std::any_cast<int>(var) << std::endl;
    std::cout << std::any_cast<std::string>(str) << std::endl;
    auto ret = std::any_cast<Person>(info);
    std::cout << "姓名: "
              << ret.name
              << " 年龄: "
              << ret.age
              << std::endl;

    return 0;
}
```
