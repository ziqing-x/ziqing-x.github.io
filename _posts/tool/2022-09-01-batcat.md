在 linux 下常常会用到 cat 命令，但是 cat 命令不带颜色且不能前后滚动或者翻页，下面推荐一个 cat 的平替 --- bat 。

## 安装

安装 batcat
```bash
sudo apt install bat
```
给 batcat 设置别名

```bash
# 打开配置文件
vim ~/.bashrc
# 在最后添加下面这一句
alias "bat=batcat"
# 保存退出后使命令生效
source  ~/.zshrc
```


## 使用

使用和 vim 类似

### 1、查看文件内容

```bash
bat <filename>
```

### 2、停止查看

```bash
：q
```

### 3、向前翻滚

```bash
ctrl + f
```

### 4、向后翻滚

```bash
ctrl + b
```

### 5、搜索关键字

```bash
/关键字
```
